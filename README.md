# Monopoly Web Online
> Enjoy the classic Monopoly with your friends on the Web

[![Build Status][travis-image]][travis-url]
[![Coveralls Status][coveralls-image]][coveralls-url]

Play Web version Monopoly online

![](monopoly.png)

## Installation

```sh
cd monopoly
npm install
cd app
npm install
```

## Development setup

Run unit tests 
```sh
npm test
```
